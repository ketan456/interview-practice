Angular Pipes

A Pipe in Angular is used to transform data before displaying it in a template it allows you to format filter or modify dat dynamically without changing the original value

Built-In Pipes

Ready-to-use pipes for common tasks like text formatting, date, numbers, JSON, etc.

  • uppercase
  • currency
  • date
  • percent

Async Pipe

Automatically handles observables or promises in templates.

Example: (auto-subscribes and unsubscribes).

Custom Pipes

User-defined for specific transformations.

Example: A custom pipe to capitalize words in a string.

Real Example:{{ "ketan" | capital }}

Pure Pipes

Only recalculates if the input value changes.

All built-in pipes are pure (default).

Run only when data refrerence change

optimized for performance

Good for statice value like string, number and date

Impure Pipes

Recalculates every change detection cycle (useful for dynamic data).

run on every change detection cycle

useful for dynamically changing data but can impact on performance

Example: Defined as pure: false in the pipe's decorator.
