Angular Directives

A directive in Angular is a special class that allows you to add behavior to element components on atributs in the dom

Directive help modify the appearnce straucture behaviour of dom elements dynamically

Structural Directives

Structural Directives are directives which change the dom layout by adding and removing dom elements

  • *ngIf: Conditionally adds or removes elements.
  • *ngFor: Loops over a collection.
  • *ngSwitch: Displays elements conditionally based on a value.

Attribute Directives

Change the appearance or behavior of Dom elements and angular components with attribute directives

  • ngClass: Dynamically assigns CSS classes.
  • ngStyle: Dynamically applies styles.
  • customDirective: User-defined directive for extended functionality.

Component Directives

Custom components that include logic and templates.

  • Each Angular component is a directive with a template.
  • Defined using the Component decorator.
  • Used to create reusable UI elements.
