RxJS Subjects

A Subject in RxJS is a special type of Observable that allows values to be multicasted to multiple Observers. Unlike a normal Observable, a Subject acts as both an Observable (emits values) and an Observer (can receive values).

  • Acts as both Observable & Observer – You can subscribe to it and also manually emit values.
  • Multicasting – The same emitted value is shared among all subscribers.
  • Subscribers receive only future values – They do not get previously emitted values.

BehaviorSubject in RxJS (Angular)

A BehaviorSubject is a special type of RxJS Subject that:

Requires an initial value when created. Always stores the last emitted value. New subscribers receive the last emitted value immediately.

Why Use BehaviorSubject?

  • Great for managing application state (e.g., authentication status, theme settings, user data). x
  • Ensures new subscribers get the latest state immediately.
  • Acts as both an Observable and an Observer (can next() new values).
